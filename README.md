# A股行情展示系统

## 📊 系统功能

- A股实时行情展示
- K线图和成交量分析
- 市场指数概览（上证、深证、创业板）
- 中文名称搜索股票
- 历史数据查询

## 🔧 技术栈

- **前端框架**: Streamlit
- **数据可视化**: Plotly
- **数据处理**: Pandas
- **数据来源**: 新浪财经API

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run stock_app.py
```

### 3. 访问地址

- 本地访问：http://localhost:8501
- 局域网访问：http://你的IP:8501

## 📝 功能说明

### 市场概览
- 实时显示上证指数、深证成指、创业板指
- 卡片式展示，涨跌一目了然
- 自动刷新数据（缓存1分钟）

### 股票查询
- 支持中文名称搜索（如：贵州茅台）
- 支持股票代码搜索（如：600519）
- 实时联想匹配，快速选择
- 热门股票快速选择

### 数据展示
- 实时价格、涨跌幅
- K线图和成交量
- 今开、最高、最低价
- 历史数据表格

## 💡 使用技巧

1. **搜索股票**
   - 在侧边栏输入框输入股票名称或代码
   - 系统自动显示匹配结果
   - 点击选择即可查询

2. **快速选择**
   - 使用"快速选择热门股票"下拉框
   - 一键查询常用股票

3. **日期范围**
   - 支持1个月、3个月、6个月、1年
   - 查看不同时间段的历史走势

4. **清除缓存**
   - 点击"清除缓存"按钮
   - 获取最新实时数据

## 📊 数据说明

### 数据来源
- 所有数据来自新浪财经API
- 实时行情数据（缓存1分钟）
- 历史K线数据（缓存5分钟）

### 股票代码格式
- 上海主板：600519.SS（贵州茅台）
- 深圳主板：000001.SZ（平安银行）
- 创业板：300750.SZ（宁德时代）
- 科创板：688981.SS（中芯国际）

### 数据更新频率
- 市场指数：每分钟更新
- 股票实时行情：每分钟更新
- 历史K线：每5分钟更新

## ⚠️ 注意事项

1. **网络要求**
   - 需要能访问新浪财经API
   - 建议使用稳定的网络连接

2. **数据准确性**
   - 数据来自新浪财经，与同花顺一致
   - 实时数据可能有几秒延迟

3. **使用限制**
   - 仅支持A股查询
   - 不支持美股、港股

## 🔧 故障排除

### 问题1：无法获取股票列表
**解决方案**：
- 检查网络连接
- 点击"清除缓存"重试
- 系统会自动降级到常用股票列表

### 问题2：股票数据获取失败
**解决方案**：
- 检查股票代码是否正确
- 确认是A股股票
- 稍后重试

### 问题3：搜索无结果
**解决方案**：
- 输入更精确的关键词
- 尝试输入股票代码
- 使用快速选择功能

## 📞 技术支持

如有问题，请检查：
1. 网络连接是否正常
2. 股票代码格式是否正确
3. 是否为A股股票
4. 尝试清除缓存重试
