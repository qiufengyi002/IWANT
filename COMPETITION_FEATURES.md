# 🏆 AI竞赛项目说明

## 项目名称
A股行情展示与模拟交易系统

## 项目简介
这是一个基于Streamlit开发的A股行情查询和模拟交易平台，集成了实时行情数据、技术分析图表、模拟交易、持仓管理等功能，为用户提供完整的股票投资学习和模拟体验。

## 核心功能

### 1. 实时行情查询 📈
- 支持3000+只A股股票查询
- 实时价格、涨跌幅显示
- 市场指数概览（上证、深证、创业板）
- 智能搜索（支持股票代码和中文名称）
- K线图和成交量分析
- 历史数据查询（1个月-1年）

### 2. 模拟交易系统 💰
- 虚拟资金交易（初始100万元）
- 买入/卖出操作
- 实时价格获取
- 资金和持仓检查
- 交易记录追踪

### 3. 持仓管理 📊
- 实时持仓查看
- 盈亏计算（已实现+未实现）
- 持仓分布饼图
- 盈亏排行榜
- 交易统计分析

### 4. 自选股管理 ⭐
- 添加/移除自选股
- 实时价格监控
- 快速查看详情
- 涨跌幅实时更新

### 5. 交易记录 📜
- 完整的交易历史
- 交易统计（买入/卖出次数、金额）
- 交易筛选（按类型、股票）
- 卡片式展示

## 技术架构

### 前端框架
- **Streamlit**: 快速构建数据应用
- **Plotly**: 交互式图表可视化
- **HTML/CSS**: 自定义UI美化

### 后端技术
- **Python**: 核心开发语言
- **SQLite**: 轻量级数据库
- **Pandas**: 数据处理和分析
- **Requests**: API数据获取

### 数据来源
- **新浪财经API**: 实时行情数据
- 数据准确性与同花顺一致
- 支持A股全市场数据

### 数据库设计
```sql
-- 用户表
users (user_id, username, cash, created_at)

-- 持仓表
positions (position_id, user_id, stock_code, stock_name, quantity, avg_cost)

-- 交易记录表
transactions (transaction_id, user_id, stock_code, stock_name, type, quantity, price, amount, created_at)

-- 自选股表
watchlist (watchlist_id, user_id, stock_code, stock_name, created_at)
```

## 技术亮点

### 1. 性能优化
- 数据缓存机制（1分钟TTL）
- 减少API请求次数
- 提升响应速度

### 2. 状态管理
- Session State管理用户状态
- URL参数保持登录状态
- 页面切换状态同步

### 3. 用户体验

- 渐变色和视觉反馈
- 响应式布局

### 4. 数据准确性
- 准确的盈亏计算（FIFO算法）
- 时区转换（UTC→本地时间）

### 5. 错误处理
- API请求失败降级处理
- 用户输入验证
- 友好的错误提示

## 部署说明

### 本地运行
```bash
# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run stock_app.py
```

### Docker部署
```bash
# 构建镜像
docker build -t stock-app .

# 运行容器
docker run -p 8501:8501 stock-app
```
